# Sankaku

円盤の上で三角形を描くプログラム

## 使い方 (HTML に読み込む場合)

1. HTML ファイルで canvas の id を "sankaku" とします。

```
<canvas id="sankaku" width="200" height="200"></canvas>
```

2. HTML ファイル内で Sankaku.js を読み込みます。
 
```
<script type="text/javascript" src="Sankaku.js"></script>
```
